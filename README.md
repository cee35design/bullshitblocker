## installation
because i do not possess the 5$ required for a google chrome webstore account, we'll have to do it manually

### step 1
download the `.crx` file from releases

### step 2
go to `chrome://extensions` on chrome (or whatever chromium-based browser you use)

### step 3
enable the "developer mode" toggle on the top right of the screen

### step 4
drag and drop the downloaded `.crx` file on the window and accept whatever it says

### step 5
youre done lmao
